# Portfolio Project

## Overview
A premium, enterprise‑level portfolio built with **React + Vite** on the frontend and **Supabase** as the backend (PostgreSQL, Auth, Storage). It features modern 3D animations, a collapsible sidebar, and a full admin control panel for managing content, SEO settings, and files.

## Features
- Collapsible sidebar with Home, About, Skills, Projects, Contact, plus Login/Register or Control Panel based on auth state.
- Admin dashboard with panels for Profile, About, Skills, Projects, SEO, and File Management.
- 3D particle sphere background for a premium visual experience.
- Dynamic data fetching from Supabase; graceful fallback to static data.
- SEO management via Admin panel (meta title, description, keywords).

## Setup & Development
```bash
# Clone the repo
git clone <repo-url>
cd portfolio/frontend

# Install dependencies
npm install

# Create .env.local with Supabase vars (prefix VITE_)
# Example:
# VITE_SUPABASE_URL=https://your-project.supabase.co
# VITE_SUPABASE_PUBLISHABLE_KEY=public-anon-key

# Development server
npm run dev   # http://localhost:5174 (or next free port)

# Build for production
npm run build
```

## Deployment
Deploy the `dist` folder to Vercel (or any static host). Add the same `VITE_` environment variables in the Vercel dashboard.
